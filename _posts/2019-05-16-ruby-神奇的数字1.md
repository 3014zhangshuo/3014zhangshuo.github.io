ruby中变量`$1`来储存capture group`(...)`捕获到的内容，例如：

```ruby
"foobar" =~ /foo(.*)/
# => 0
$1
# => "bar"
```

如果不想储存的话，可以使用non-capturing group，例如：

```ruby
"foobar" =~ /foo(?:.*)/
# => 0
$1
# => nil
```

与`$1`相对的还有一个`\1`，但只用于`sub`和`gsub`方法，例如：

```ruby
"foobar".sub(/foo(.*)/, '\1\1')
# => "barbar"

# 还可以这么用
"foobarfoo".sub(/foo(bar)(foo)/, '\1\2\2')
# => "barfoofoo"
```
